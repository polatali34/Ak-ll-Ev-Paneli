#  Akıllı Ev & Gömülü Sistem Kontrol Paneli (ASP.NET Core MVC)

Bu proje, bir gömülü sistemi web üzerinden kontrol etmek ve durumunu izlemek amacıyla geliştirilmiş modern bir kontrol paneli (dashboard) simülasyonudur.

##  Projenin Amacı
Gömülü sistemler ile web uygulamalarının nasıl entegre çalışabileceğini göstermek. Şu an için arayüz (Frontend) ve iş mantığı (Backend) tamamlanmış olup, donanım entegrasyonuna hazır bir altyapı sunmaktadır.

##  Kullanılan Teknolojiler

*   **Backend:** C#, ASP.NET Core MVC
*   **Frontend:** HTML5, CSS3, JavaScript (Geri Sayım/Döngü Mantığı)
*   **Tasarım Çerçevesi:** Bootstrap 5
*   **İkonlar:** Bootstrap Icons

##  Öne Çıkan Özellikler

*   **Dinamik Veri Akışı (MVC Mimarisi):** C# Controller üzerinden View'a veri aktarımı.
*   **Duruma Dayalı Stil (Conditional Styling):** Sistem açık/kapalı durumuna göre anlık değişen renkler, ikonlar ve butonlar (örn. Motor çalışırken mavi, dinlenirken sarı).
*   **JavaScript Zamanlayıcı:** Sayfa yenilenmeden çalışan, kullanıcı tanımlı 5 dakikalık (veya özel girilen) endüstriyel döngü motoru simülasyonu.
*   **Responsive Tasarım:** Mobil cihazlar ve masaüstü ekranlar ile tam uyumlu UI.
